SmoothApproximations.approx_heaviside — Method
approx_heaviside(; from=0, to=1, at=0, k=50)

Approximate the heaviside step function, generalized to any step size (from and to) and location (at), using a hyperbolic tangent function with steepness parameter k.

  • Heaviside:
x < atx == atx > at
$from$$(to-from) / 2$$to$
  • Approximate Heaviside:

\[(to - from) * tanh(k * (x - at)) / 2 + (from + to) / 2\]
